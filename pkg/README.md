# pkg

This directory hosts reusable infrastructure packages for the scaffold.

Current packages fall into two groups:

- runtime infrastructure used by the main app container
- tool-oriented helpers such as `sqlgen` and `yaml2go`

The repository currently includes:

- `cache`
- `cli`
- `crypto`
- `database`
- `dbtx`
- `executor`
- `httpserver`
- `i18n`
- `jwt`
- `logger`
- `rbac`
- `sqlgen`
- `storage`
- `utils`
- `yaml2go`
